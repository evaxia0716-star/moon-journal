## 這包裡有什麼

| 檔案 | 角色 | 比喻 |
|---|---|---|
| index.html | 主程式（畫面＋邏輯全在裡面） | 整間店 |
| manifest.json | PWA 身分證 | 告訴手機「我是 App」 |
| sw.js | Service Worker | 駐店店員，把店存一份在手機裡，沒網路也開得了門 |
| icon-192.png / icon-512.png / icon-maskable-512.png | 桌面圖示 | 大頭照（Android 用） |
| apple-touch-icon.png | 桌面圖示 | 大頭照（iPhone 用） |

六個檔案要放在**同一層資料夾**，不能分開。

## 第一步：上架到 GitHub Pages（你做過的流程）

1. 到 GitHub 建一個新 repo，例如 `moon-journal`（建議設 **Private 不行**——GitHub Pages 免費版要 Public repo 才能用。介意的話 repo 名取得低調一點即可，反正資料不在裡面，程式碼公開也看不到任何人的紀錄）
2. 把這 6 個檔案全部上傳到 repo 根目錄
3. repo → Settings → Pages → Source 選 `main` branch、`/ (root)` → Save
4. 等 1–2 分鐘，網址會是：
   `https://你的帳號.github.io/moon-journal/`

> 跟「台北有影 3」的差別：那邊資料存在 Google Sheets（雲端倉庫），這邊資料存在**每個人自己的手機瀏覽器**（localStorage）。所以不用 Apps Script、不用任何後端——這就是它隱私的原因，也是它的限制：換手機資料不會跟過去，要用「設定 → 匯出備份」搬家。

## 第二步：安裝到手機

### iPhone（一定要用 Safari）
1. 用 **Safari** 開啟網址（Chrome 不行，iOS 限制）
2. 點下方中間的「分享」按鈕（方框加箭頭）
3. 往下滑，點「**加入主畫面**」
4. 桌面出現月記圖示，點開就是全螢幕 App，沒有瀏覽器網址列

### Android（用 Chrome）
1. 用 Chrome 開啟網址
2. 通常會自動跳出「安裝月記」的提示；沒有的話點右上「⋮」→「**加到主畫面**」或「**安裝應用程式**」
3. 完成後桌面出現圖示

安裝後**離線也能開**——第一次開啟時店員（sw.js）已經把整間店搬進手機了。

## 改版時要注意的一件事

以後改了 index.html 重新上傳，手機上的 App 可能還是顯示舊版——因為店員手上拿的是舊存貨。解法：打開 `sw.js` 第一行，把

```
const CACHE_NAME = 'moon-journal-v3';
```

版號加一（每次改版都要，目前已是 v4）。店員看到版號變了，就會自動丟掉舊存貨、進新貨。

## 功能總覽（v2.0 基礎）

**今日**
- 環形週期圖（四階段＋呼吸金點）、下次經期預估、今日紀錄縮影
- 回憶卡：「去年的今天」「30 天前的你」自動浮現
- 備份提醒橫幅：累積 30 筆未備份紀錄時出現，一鍵匯出

**紀錄**：經血量／症狀／心情／精力／自由筆記，歷史時間軸

**洞察**
- 連續紀錄天數（最長紀錄、下一枚徽章進度）
- 月報卡：◀ ▶ 切換月份，看紀錄天數、平均精力、最常心情／症狀、精力曲線
- 成就牆 **30 項**，分四層：
  - 堅持層 ×6（連續紀錄）：🌱3｜🌙7｜🌸21｜🌕66｜🌟100｜🌌365 天
  - 里程層 ×5（累積週期）：🌒1｜🌓3｜🌔6｜🌕12｜🌖24 個（跟著月相一路變圓）
  - 探索層 ×10：第一篇筆記、第一次備份、搬家成功（匯入）、走過四個階段、五項全填、第一籤、籤友（抽 30 次）、五十篇、百日誌、症狀圖鑑、心情光譜
  - 彩蛋層 ×9：達成前只顯示「？？？」，遇到了才知道是什麼 🎁
- 自我認識解鎖：1 週期→基本預測｜3→PMS 模式｜6→精力高峰圖｜12→年度報告

**設定**
- 週期參數、生日（某個彩蛋需要它）
- 永久儲存狀態顯示（App 啟動時自動向系統申請，不讓 iOS 自動清資料）
- 上次備份時間＋未備份筆數、匯出／匯入備份、清除資料

## 資料保險三道鎖

1. **永久儲存**：啟動時自動申請 `navigator.storage.persist()`，設定頁可看狀態
2. **備份提醒**：每累積 30 筆新紀錄，主畫面跳橫幅提醒匯出
3. **備份成就化**：第一次匯出解鎖「🛡️ 資料守護者」——用遊戲機制保護資料

> v1 的舊資料可直接沿用，升級 v2 不會弄丟任何紀錄。
